# faq-center Project

=== **Project Description** ===
* faq-center is a contact center web application.
* faq-center project was created using spring framework and thymeleaf view template.

=== **How to run it** ===

1. gradle refresg
2. run "faq-center\src\main\java\FAQ\faqcenter\FaqCenterApplication"
